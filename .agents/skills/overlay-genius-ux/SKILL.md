# Overlay Genius UX Skill

Use this skill when working on overlay UI.

## UX requirements

- The overlay must reduce fatigue, not add it.
- Compact mode is short.
- Deep context opens on click/hotkey.
- Never block player choices or checks.
- Keep original-first playthrough intact.
- Support debug mode separately from player mode.

## Card structure

1. Ukrainian translation.
2. Tiny explanation icon.
3. Expandable annotation:
   - idiom,
   - sarcasm,
   - tone,
   - skill voice,
   - cultural equivalent,
   - audio note.
