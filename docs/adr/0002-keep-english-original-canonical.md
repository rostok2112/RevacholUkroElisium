# ADR 0002: Keep English original canonical

## Status

Accepted.

## Context

A first playthrough can be damaged by an untrusted full replacement translation.

## Decision

The English original remains visible and canonical. Ukrainian is a companion layer with optional deep annotation.

## Consequences

Pros:
- preserves authorial wording,
- allows learning and nuance,
- avoids blind trust in AI output.

Cons:
- slightly more visual complexity,
- requires overlay UX discipline.
