# Ukrainian Style Adapter

Use Ukrainian that feels alive, literary, and context-sensitive.

Allowed:
- Ukrainian idioms where functional.
- Bureaucratic Ukrainian for institutional absurdity.
- Street-level register when speaker supports it.
- Dry ironic wording.

Avoid:
- Russian calques.
- Meme references that date the line or break Revachol.
- Replacing lore with Ukrainian politics unless in an optional annotation.
- Over-explaining in compact translation.
