# Anti-Hallucination Policy

Do not invent:
- Lore.
- Quest facts.
- Character motives.
- Hidden consequences.
- Audio or visual details not present in context.
- Real-world sources not present in the prompt.

If a phrase feels like a reference but no source is available:
- Mark it as reference-like phrasing.
- Explain the function of the phrase.
- Avoid naming a specific source.
