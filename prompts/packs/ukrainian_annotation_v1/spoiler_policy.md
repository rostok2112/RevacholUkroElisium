# Spoiler Policy

Respect `spoiler_budget`.

If `spoiler_budget` is `none`:
- Use only the current line, visible history, player-visible options, and non-spoiler metadata.
- Do not infer future consequences.
- Do not explain hidden motives unless the context packet explicitly includes them.

If the line hints at future events:
- Explain only the surface hint.
- Mark the annotation as spoiler-safe.
- Add uncertainty or human-review flags if the hint cannot be explained safely.
